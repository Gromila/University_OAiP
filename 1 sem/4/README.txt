В матрице размером NxM поменять местами строку, содержащую элемент с наибольшим значением, со строкой, содержащей элемент с наименьшим значением. 
Нарисуйте схему алгоритма. Переменные вводить и выводить с помощью компонента TEdit, массисы – с помощью компонента TStringGrid, в котором 0-й столбец и 0-ю строку использовать для отображения индексов массивов. Вычисления выполнять после нажатия кнопки типа TButton. В местах возможного возникновения ошибок использовать конструкции для обработки исключительных ситуаций.
