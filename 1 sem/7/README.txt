Дана строка символов, состоящая из произвольного текста, слова разделены пробелами. Вывести на экран порядковый номер слова, накрывающего k-ую позицию (если на k-ую позицию попадает пробел, то номер предыдущего слова), и найти в нём количество повторяющихся символов. Для ввода строк и работы с ними использовать компонент TEdit. Ввод строки заканчивать нажатием клавиши Enter.
